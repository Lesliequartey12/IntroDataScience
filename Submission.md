Data Science Training and Tools Overview 

<!-- Python-guide  -->
Python is a programming language that was designed to read almost like plain English. 

<!-- Marimo-notebooks  -->
Marimo is a reactive Python notebook that runs as pure Python files. Marimo notebooks can work with any python tool.

<!-- Git-version-control  -->
Git is used by professionals to help collaborate with others on the same project. With git every progress is saved. 

<!-- Plotly-visualization  -->
Plotly is a visualization library that creates interactive, mobile and user friendly professional charts.

<!-- Polars-data-wrangling -->
Polars is a DataFrame library for Python that's designed for speed and efficiency and has clear and readable syntax. 

<!-- Vscode-guide  -->
Visual Studio Code is a free code editor made by Microsoft that comes with a built in terminal. 

<!-- Uv-package-management -->
UV is a modern Python package manager written in Rust that's designed to be fast and reliable. It ensures you have the correct python version. 


<!-- Data Science -->
Data science involves digging into data to uncover insights and knowledge by working around statistics, coding, expertise in a field, and sharp analysis. Basically, it turns messy data into useful informations that help organizations make better decisions. 

Data scientists collect messy data e.g CSV files, explore data to find patterns, build predictive models and communicate their findings to stakeholders who can understand and make decisions. 

Healthcare, finance, manufacturing and retail sectors are practical industries that implements data findings gathered from data scientists. 

<!-- Markdown-writing -->
Markdown is a simple set of symbols you type into plain text to control how it looks when displayed.
Markdown is used everywhere. GitHub, Reddit, Notion, Slack, WhatsApp, Discord, Jupyter Notebooks, documentation sites, blogs, and more. 

Markdown writing is basically faster than reaching for your mouse to click formatting buttons, it
Looks clean and professional with zero design effort
Easy to learn in one sitting.
